# Separators

Twelve tokens, formed from ASCII characters, are the separators (punctuators).
````
Separator:
    (one of)
        ( ) { } [ ] ; , . ... @ ::
````

# Operators

````
Operator:
    (one of)
        = > < ! ~ ? : ->
        == >= <= != && || ++ --
        + - * / & | ^ % << >> >>>
        += -= *= /= &= |= ^= %= <<= >>= >>>=
````
